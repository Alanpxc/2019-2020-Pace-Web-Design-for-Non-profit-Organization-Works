The Element of HTML


The elements in html is the whole html tags, which contains texts and medias. In the elements of HTML, the tags always start from the start tag (opening tag) and end tag (closing tag).


For example:

<.h1>Hey, this is the element of html page!<./h1>

From this sentence:

The . is not letting the html tags activate, it can be ignore. The <.h1> is the opening tag and the <./h1> is the closing tag. "Hey, this is the element of html page!" is the text in the tags. The whole things we called the element.
