Using and Linking a .css File


CSS is to customize the HTML element and decide the way to display it, the CSS file is an external way.


Example for .css file (File name is "BostonStyles.css):

body {
background-color: yellow;
}
h1 {
color: red;
}
p {
color: blue;
}

Warning: need to type<.link rel="stylesheet" href="BostonStyles.css"> (Ignore the . at the front) in the header
